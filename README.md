# Pei Ling Chang — AI Systems Portfolio

Personal portfolio website for **Pei Ling Chang**, hosted with GitHub Pages.

## Live site

After GitHub Pages is enabled, the website will be available at:

**https://melody-taiwan.github.io/**

## Files

- `index.html` — portfolio homepage
- `resume.html` — web resume with print-to-PDF support
- `styles.css` — responsive styling and light/dark themes
- `script.js` — theme toggle, mobile navigation, and reveal animations
- `favicon.svg` — site icon

## Deployment

In the repository, open **Settings → Pages** and select:

- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

Then save and wait for GitHub Pages to publish the site.
